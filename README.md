# docker-go-server
HTTP go server for docker's container.

## Requirements

- [Docker](https://docs.docker.com/install/#supported-platforms) >= 18.09.6
- [Docker compose](https://docs.docker.com/compose/install) >= 1.24.0

## Install

```bash
make install
```

Open http://localhost:8080 

## Commands

Connect to shell
```bash
make shell
```

Show logs
```bash
make logs
```

Stop application
```bash
make stop
```
